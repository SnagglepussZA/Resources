変数

javascript、そして他な「スクリプトイング言語」も、変数の使い方は他な言語の種類より簡単だ。変数をコードで作るときは var (variableの略)で宣する　（Declare）。

例えば：var age = 15;
と書いたら、パソコンに、「"age" (年齢)という変数が存在してるよ、価値は15を入れてください」と伝えてるようなもの。 スクリプト以外は普通に先にパソコンに変数は何のデータの種類を持つのか教えないといけない。

例として、このページのスクリプトタグの中に以下をやろう：
var price = 150;
var money = 1000;
var count = 1000;
price (値段)という変数を宣して、150の価値を入れる
money (お金)という変数を宣して、1000の価値を入れる
count (数)という変数を宣して、0の価値を入れる

トマト何個持ってる？　

ひとつ買うボタン押したら、お金から値段の価値が引かれて、カウントは一つ増える。なんで変数いるのかと思えば、とりあえず データを覚えるために必要なんだ。

上のことをやるために、ファンクション（関数）を使ってる

function showExample (){
document.getElementById ("price").innerHTML = "" + price + "円";
document.getElementById ("money").innerHTML = "" + money + "円";
document.getElementById ("count").innerHTML = "" + count;
}

function buy1 (){
money = money - price;
count = count + 1;
showExample ();
}
